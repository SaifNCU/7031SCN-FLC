# 🧠 Kidney Disease Risk Prediction using Fuzzy Logic + Genetic Algorithm (TASK B)

This project implements a **Fuzzy Inference System (FIS)** enhanced with a **Genetic Algorithm (GA)** to optimise membership functions for improved kidney disease risk prediction.

---

## 🚀 Overview

- 📊 Fuzzy Logic → Models uncertainty  
- 🧬 Genetic Algorithm → Optimises parameters  
- 📉 Goal → Minimise error (MSE & MAE)

---

## 📂 Project Structure

```
TASK_B/
│── main_ga_eval.m
│── kidney_fis.fis
│── kidney_optimized_fis.fis
│── README.md
```
---

## ▶️ How to Run

Open MATLAB and run:

```
    cd('path_to_project')
    main_ga_eval
```
